Sobre o projeto:

O projeto tem o intuito de usar informações providas de APIS públicas, processar elas e transformar em dados uteis para mostrar em gráficos do Google Charts

Objetivo:

Usar os ensinamentos de Google Charts junto com as informações de API e integrar os dois usando JavaScript em um HTML dinâmico

Linguagens & ferramentas utilizadas:

  • API
  • CSS3
  • HTML5
  • JS
  • Google Charts
